<h2>🛠️ Installation Steps:</h2>

<p>1. Runs the app in the development mode.</p>

```
yarn start
```

<p>2. Launches the test runner in the interactive watch mode.</p>

```
yarn test
```

<p>3. Builds the app for production to the build folder.</p>

```
yarn build
```
