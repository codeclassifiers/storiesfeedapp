<h2>🛠️ Installation Steps:</h2>

<p>1. Run the app in development mode</p>

```
yarn start
```

<p>2. Launch the test runner in the interactive watch mode.</p>

```
yarn test
```

<p>3. Build the app for production to the build folder.</p>

```
yarn build
```
